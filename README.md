## Drive me
Mobile app for Carsharing, that allows us to see list of cars, their information, book car for rent choosing comfortabe dates,
or choose dates, we need and see available cars at this time. Authentication via Supabase Auth, interactive map, where user can see locations
where it's possible to pickup/drop-off car, allowed driving zone.
This project is developing now, the Client Panel is ready, now I'm focused on Admin Panel, where he can add/delete/edit car informations,
approve/reject bookings, edit locations on map interactively or manual.
